PL-300 Eight-Week Learning Plan (1 Hour Per Day)

Study one hour per day, five days per week. Each week includes sugested Microsoft Learn modules, a Kaggle dataset to download, a use-case assignment, and focused daily tasks. By Week 8, you will complete a full end-to-end BI solution aligned with PL-300 exam expectations.


Kaggle Datasets to be Used

Week 1: Superstore Sales
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final (kaggle.com in Bing)

Week 2: Retail Sales Forecasting
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting (kaggle.com in Bing)

Week 3: HR Analytics
https://www.kaggle.com/datasets/rhuebner/human-resources-data-set (kaggle.com in Bing)

Week 4: Financial Transactions
https://www.kaggle.com/datasets/ealaxi/paysim1 (kaggle.com in Bing)

Week 5: Marketing Campaign
https://www.kaggle.com/datasets/jackdaoud/marketing-data (kaggle.com in Bing)

Week 6: E-commerce Customers
https://www.kaggle.com/datasets/carrie1/ecommerce-data (kaggle.com in Bing)

Week 7: Global Store Sales
https://www.kaggle.com/datasets/kyanyoga/sample-sales-data (kaggle.com in Bing)

Week 8: Choose any dataset from above for the capstone project.


Ten Practice Exercises

1. Import a CSV dataset and create a star schema with at least one dimension table.
2. Build a date table using DAX and mark it as a date table.
3. Create five DAX measures: Total Sales, Total Quantity, Profit Margin, Year-over-Year Sales, and Percent of Total.
4. Build a slicer panel with synced slicers across pages.
5. Create a tooltip page with dynamic visuals.
6. Build a drill-through page for product-level analysis.
7. Use Power Query to split, merge, and clean columns.
8. Create a KPI visual with conditional formatting.
9. Publish a report to Power BI Service and configure a workspace.
10. Create a dashboard with pinned visuals and set up a data alert.


Weekly Use-Case Assignments

Week 1: Build a basic sales overview dashboard for a retail manager.
Week 2: Create a forecasting report showing expected sales for next quarter.
Week 3: Build an HR attrition analysis dashboard.
Week 4: Detect anomalies in financial transactions and highlight risk.
Week 5: Build a marketing funnel performance dashboard.
Week 6: Analyze customer purchase behavior and segment customers.
Week 7: Build an executive sales performance report with drill-through.
Week 8: Capstone project: Create a full end-to-end BI solution using any dataset.


Full Eight-Week Study Plan (1 Hour Per Day)

Week 1: Power BI Foundations (Superstore Dataset) Microsoft Learn: Get started with Microsoft data analytics
https://learn.microsoft.com/training/paths/get-started-data-analytics/ 

Get started with Power BI
https://learn.microsoft.com/training/modules/get-started-power-bi/ 

Daily Plan: Day 1: Power BI overview and interface
Day 2: Import Superstore dataset and explore fields
Day 3: Build basic visuals (bar, line, table)
Day 4: Create your first report page
Day 5: Weekly use-case assignment


Week 2: Power Query and Data Cleaning (Sales Forecasting Dataset) Microsoft Learn: Clean, transform, and load data in Power BI
https://learn.microsoft.com/training/paths/clean-data-power-bi/ 

Daily Plan: Day 1: Power Query basics
Day 2: Column transformations
Day 3: Merge and append queries
Day 4: Data profiling and data quality
Day 5: Weekly use-case assignment


Week 3: Data Modeling (HR Analytics Dataset) Microsoft Learn: Model data in Power BI
https://learn.microsoft.com/training/paths/model-data-power-bi/

Daily Plan: Day 1: Star schema fundamentals
Day 2: Relationships and cardinality
Day 3: Hierarchies and role-playing dimensions
Day 4: Build a date table
Day 5: Weekly use-case assignment


Week 4: DAX Fundamentals (Financial Transactions Dataset) Microsoft Learn: Use DAX in Power BI Desktop
https://learn.microsoft.com/training/modules/dax-power-bi/ 

Daily Plan: Day 1: Basic measures (SUM, COUNT, DISTINCTCOUNT)
Day 2: CALCULATE and filter context
Day 3: Time intelligence (YTD, YoY, MoM)
Day 4: KPIs and conditional formatting
Day 5: Weekly use-case assignment


Week 5: Visualizations (Marketing Dataset) Microsoft Learn: Visualize data in Power BI
https://learn.microsoft.com/training/paths/visualize-data-power-bi/ 

Daily Plan: Day 1: Choosing the right visual
Day 2: Slicers, filters, bookmarks
Day 3: Tooltips and drill-through
Day 4: Page navigation and design
Day 5: Weekly use-case assignment


Week 6: Power BI Service (E-commerce Dataset) Microsoft Learn: Introduction to Power BI Service
https://learn.microsoft.com/training/modules/introduction-power-bi-service/

Daily Plan: Day 1: Publish to Power BI Service
Day 2: Workspaces and permissions
Day 3: Dashboards and alerts
Day 4: Data refresh and gateways
Day 5: Weekly use-case assignment


Week 7: Exam-Style Scenarios (Global Sales Dataset) Microsoft Learn: Get data with Power BI Desktop
https://learn.microsoft.com/training/modules/get-data-power-bi/ 

Daily Plan: Day 1: Build a full model from scratch
Day 2: Create ten measures
Day 3: Build three report pages
Day 4: Publish and create a dashboard
Day 5: Weekly use-case assignment


Week 8: Capstone Project and Exam Readiness Daily Plan: 
Day 1: Choose dataset and define business questions
Day 2: Clean and model data
Day 3: Build measures
Day 4: Build visuals and publish
